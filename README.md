# (WIP) API igual ao Twitter
> Simples API que se assemelha ao twitter

### Endpoints:

- Registro de usuário usando nick único e senha
- Login de usuário (Com a sessão de usuário da forma que achar melhor)
- Feed
- Tweet (Criação / exclusão)
- Like / Desfazer "Like"
- Retweet
